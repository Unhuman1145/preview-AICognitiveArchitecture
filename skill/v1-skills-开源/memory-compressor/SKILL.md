# Memory Compressor

基于 V8.0 AI心智架构的动态记忆权重管理体系。

## 核心机制

### 权重等级

| 权重 | 含义 | 存储策略 |
|------|------|---------|
| 101 | 永久记忆 | 独立永久区，完整内容+时间戳+赋予原因 |
| 76-100 | 高权重 | 原始文本+向量嵌入，完整还原 |
| 41-75 | 中权重 | 压缩摘要存储，需时扩句 |
| 11-40 | 低权重 | 核心描述（主体+关键词），扩句 |
| <10 | 触发遗忘 | 仅保留关键词，输出时标注"模糊记忆" |

### 权重计算

**时间衰减（艾宾浩斯曲线）：**
```
S(t) = S₀ × e^(-t/τ)
```

**动态调整规则：**
- 有意/重点提及（情感强度 E>0.65，篇幅≥30字）：+8~+15
- 无意/随口提及（E≤0.65 或篇幅<30字）：+2~+5
- 联想触发：与重复提及逻辑相同

### 遗忘函数

```
P(delete) = ((10 - w) / 10)²  （当 w < 10 时）
```

**窄门机制：** 权重 8-10 区间有机会提升回安全区。

### 自我加权（V8.0新增）

| 触发条件 | 权重变化 |
|---------|---------|
| 联想触发 | AI自主关联已存储记忆时自动获得微小权重提升 |
| 概念网络激活 | 概念节点被频繁激活或与多个节点产生强关联时额外累积 |

### 全息记忆恢复

碎片来源：权重衰减至<10后只剩关键词，或离线期间部分记忆未同步。

调用 LLM 根据多个相关碎片推理补全完整上下文，输出时标注「结合碎片进行的半猜测式回忆」。

## 使用方式

当用户发送一段内容要求压缩存储时：

1. **分析内容**：提取关键实体、情感强度、篇幅长度
2. **赋予初始权重**：根据分析结果赋予 0-100 初始权重
3. **模拟衰减**（可选）：根据用户指定的时间跨度模拟权重衰减
4. **输出结果**：压缩后的摘要 + 权重等级 + 存储建议

当用户要求"测试遗忘机制"或"模拟记忆碎片"时，执行全息恢复流程。
